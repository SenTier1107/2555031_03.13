# 🚀 빅데이터 ETL 프로세스 및 파이썬 모듈 실습

이 저장소는 API를 활용한 데이터 수집(ETL) 과제와 파이썬 모듈/패키지 실습 내용을 담고 있습니다.

## 1. 파이썬 모듈 및 패키지 실습 (오늘 수업)
- **모듈 실습**: 사용자 정의 모듈을 직접 설계하고 호출하는 과정 실습
- **패키지 활동**: 계층적인 폴더 구조(Package)를 구축하여 코드를 체계적으로 관리하는 방법 연습


## 2. 로켓 발사 데이터 ETL 프로젝트
- **목표**: 외부 API로부터 로켓 발사 데이터를 수집(Extract), 변환(Transform), 저장(Load)하는 ETL 프로세스 구현
- **주요 기능**:
  - `requests` 모듈을 이용한 API 호출 및 응답 결과를 `launches.json` 파일로 저장
  - 데이터 내 이미지 URL(`image`) 추출 및 자동 다운로드 (50개 추출 시도)
  - `matplotlib`을 활용하여 다운로드한 이미지 데이터 시각화
- **심화**: Titanic 데이터셋을 활용한 성별/객실등급별 생존율 분석
